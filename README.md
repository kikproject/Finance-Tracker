# Finance-Tracker

Мы создаём приложение, которое делает учёт финансов осознанным и мотивирующим, помогая одним — копить на семейные цели, а другим — строить фундамент личной финансовой свободы.

Цель продукта:
Помочь пользователям в мониторинге средств/расходов. Помощь в оптимизации расходов и накоплении средств.
Основные функции и преимущества:
Мониторинг денежных средств
Возможность синхронизации семьи
Визуальное отображение финансов
Уникальное предложение:
Мотивация процесса накопления
Симуляция будущих финансовых последствий
